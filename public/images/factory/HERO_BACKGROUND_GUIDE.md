# Hero 區塊背景圖片指南

## 需要的圖片

Hero 區塊需要 **2 張背景圖片**，用於自動輪播效果。

### 圖片檔案名稱
1. `factory-aerial.jpg` - 工廠空拍圖（已存在）
2. `hero-bg-2.jpg` - 第二張背景圖（需要新增）

## 圖片規格建議

### 尺寸
- **建議尺寸**：1920x1080px (16:9 比例)
- **最小尺寸**：1280x720px
- **最大尺寸**：3840x2160px (4K)

### 格式
- **格式**：JPG
- **品質**：80-85% (平衡檔案大小與品質)
- **檔案大小**：建議 200KB - 500KB

### 內容建議

#### 圖片 1：工廠空拍圖 (`factory-aerial.jpg`)
- 工廠建築物空拍視角
- 展現工廠規模與現代化設施
- 適合的構圖：從上方或斜上方拍攝

#### 圖片 2：生產線或產品 (`hero-bg-2.jpg`)
建議選擇以下主題之一：
- **選項 A**：工廠內部生產線
  - 展現現代化生產設備
  - 工作人員操作機器的場景
  - 強調專業與品質

- **選項 B**：農膜產品應用場景
  - 農田中使用農膜的實際場景
  - 展現產品在實際應用中的效果
  - 強調產品價值

- **選項 C**：工廠外觀正面照
  - 工廠建築物的正面或側面
  - 展現企業形象與規模
  - 專業的企業外觀

### 視覺要求

1. **色彩**：
   - 避免過於鮮豔或對比強烈的顏色
   - 建議使用自然色調（綠色、藍色、灰色系）
   - 因為會加上綠色到藍色的漸層遮罩，原圖色彩不要太濃

2. **構圖**：
   - 重要元素放在中央或上方區域（文字會顯示在中央）
   - 避免在中央區域有過於複雜或搶眼的元素
   - 保持畫面簡潔，留白適當

3. **光線**：
   - 光線均勻，避免過暗或過亮
   - 避免強烈逆光或陰影

4. **文字可讀性**：
   - 背景不會與白色文字衝突
   - 加上漸層遮罩後，文字應該清晰可見

## 圖片來源建議

### 免費圖片資源
1. **Unsplash** (https://unsplash.com)
   - 搜尋關鍵字：factory, industrial, manufacturing, agriculture
   - 高品質免費圖片

2. **Pexels** (https://www.pexels.com)
   - 搜尋關鍵字：factory, production line, agricultural film
   - 免費商用圖片

3. **Pixabay** (https://pixabay.com)
   - 搜尋關鍵字：factory, industrial, manufacturing
   - 免費圖片庫

### 專業攝影
- 建議使用實際工廠照片
- 確保照片版權清晰
- 符合企業形象與品牌風格

## 圖片處理建議

### 後製調整
1. **亮度/對比度**：適度調整，確保加上遮罩後效果良好
2. **飽和度**：稍微降低，因為會加上彩色遮罩
3. **銳化**：適度銳化，提升清晰度
4. **裁切**：確保 16:9 比例，重要元素在中央區域

### 壓縮優化
- 使用工具如 TinyPNG、ImageOptim 壓縮
- 目標檔案大小：200-500KB
- 保持視覺品質

## 放置位置

將圖片放置在：
```
public/images/factory/hero-bg-2.jpg
```

## 測試建議

1. 在不同裝置上測試（桌面、平板、手機）
2. 確認文字在背景上清晰可讀
3. 檢查圖片載入速度
4. 確認輪播效果流暢

## 注意事項

- 確保圖片版權清晰，避免使用未授權圖片
- 保持兩張圖片的風格一致
- 定期更新圖片以保持網站新鮮感
- 考慮使用 WebP 格式以獲得更好的壓縮率
